# **App Name**: IronWill

## Core Features:

- Streak Tracker: Accurately tracks the current discipline streak and records the best streak, automatically resetting upon relapse.
- Daily Affirmation Check-in: Allows users to mark their progress for the day as 'clean' with a single click, preventing duplicate entries for the same day.
- Relapse Incident Logging: Provides a modal interface to log relapse events, capturing details such as reason (stress, boredom, social media) and time of day, for historical analysis.
- Urge Resistance Tracking: Enables users to log when they've resisted an urge, classifying its intensity as Low, Medium, or High, to build a record of self-control.
- Behavioral Analysis: Analyzes user's streak, relapse, and urge data to identify and display weekly insights, high-risk time windows, and common triggers.
- Discipline Score Calculation: Dynamically calculates and displays a comprehensive discipline score (0-100) based on accumulated user behavior metrics.
- Data Export Manager: Allows users to export their progress and behavioral data in various formats including JSON, CSV, and PDF for backup and personal review.

## Style Guidelines:

- The chosen color palette evokes inner strength and focus in a light theme. Primary color: a deep, muted blue-violet (#522999) for impactful elements and branding. Background color: a heavily desaturated, light blue-violet (#F3F2F7) providing a clean, subtle base. Accent color: a vibrant, calm blue (#4093E3) for calls to action and highlighting key information.
- Headline and large numerical display font: 'Space Grotesk' (sans-serif) for a modern, structured, and tech-savvy appearance, particularly effective for streak numbers. Body text font: 'Inter' (sans-serif) for optimal readability and a neutral, objective feel suitable for detailed insights and logs.
- Icons should be premium 3D, designed to complement the Neumorphic and Glassmorphic aesthetics, featuring soft gradients or embossed textures to achieve depth and match the consistent top-left light source.
- A multi-layered UI incorporating Neumorphism and Glassmorphism for depth and visual separation. Cards will feature rounded corners (20-30px), dual light/dark shadows, and a subtle floating effect. Modals will utilize a glass effect with blur and transparency. The layout maintains a consistent top-left light source across all elements.
- Subtle and smooth micro-interactions will enhance user experience, including a card hover-lift effect, tactile button press feedback, and fluid scale-and-fade transitions for modal openings. All animations will be quick, typically between 0.2-0.3 seconds.
